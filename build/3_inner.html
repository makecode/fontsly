Once you have downloaded the font file (ZIP format), you must run an unzipping software to extract all the files. Fonts not showing up in any program
  • If copy and pasted, make sure you copied a proper font file.
  • Check if installed font format is supported by target operating system.
  • Restart target application after installing new fonts.
  • If restarting the application fails, reboot operating system.
